Overview

This is a angular single page app that works with my node server for serving out a log of work that I have done as well as serve as a location where I can upload new work that has been completed.


Purpose:
For my employer to search through tasks that I have done

Reason:
Doing it as my first Angular side project to get a better understanding of the MEAN stack

Future plans:
To add a documented portfolio(instead of Google sheets) to display data excluding work related information while using the same database
